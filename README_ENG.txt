#Python Task Manager

#Project Description and Objective
Python Task Manager is a Python-based application designed to help users efficiently manage their daily tasks. With this application, users can add, view, mark as done, delete, and search tasks. All task data is saved and loaded from a JSON file, ensuring persistence even after restarting the program.

The primary objectives of this project are:
- To develop a user-friendly task management application in Python.
- To provide features such as adding tasks, marking tasks as done, deleting tasks, and searching for tasks.
- To use JSON files for saving and loading task data.

How to Use the Application
---- All Commands ----
command - Displays a list of all available commands in the program.
1. add - Add a new task by providing its name, description, and deadline.
2. view - View all tasks currently in the system, along with their status (In Progress or Done).
3. done - Mark a task as "Done" by providing its unique ID.
4. delete - Delete a task from the system using its unique ID.
5. search - Search and display tasks that need to be completed before a specific deadline.
6. exit - Exit the program.

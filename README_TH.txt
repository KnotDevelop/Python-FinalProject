#Python Task Manager

#คำอธิบายโปรเจกต์และวัตถุประสงค์
Python Task Manager เป็นแอปพลิเคชันที่พัฒนาโดยใช้ภาษา Python เพื่อช่วยให้ผู้ใช้สามารถจัดการงานประจำวันได้อย่างมีประสิทธิภาพ แอปพลิเคชันนี้สามารถเพิ่ม ดู ทำเครื่องหมายว่าเสร็จสิ้น ลบ และค้นหางานได้ โดยการจัดการข้อมูลทั้งหมดจะถูกเก็บไว้ในไฟล์ JSON เพื่อให้สามารถบันทึกและโหลดข้อมูลเมื่อโปรแกรมเริ่มต้นใหม่ได้

วัตถุประสงค์หลักของโปรเจกต์นี้คือ:
- พัฒนาแอปพลิเคชัน Python ที่ใช้งานง่าย
- สร้างระบบจัดการงานที่มีฟีเจอร์ต่าง ๆ เช่น การเพิ่มงาน, การทำเครื่องหมายว่างานเสร็จสิ้น, การลบงาน, และการค้นหางาน
- ใช้ไฟล์ JSON สำหรับการบันทึกและโหลดข้อมูล

## คำแนะนำในการติดตั้ง

1. **ติดตั้ง Python 3.x**: หากยังไม่มี Python ติดตั้งในเครื่อง ให้ดาวน์โหลดและติดตั้งจาก [เว็บไซต์ของ Python](https://www.python.org/downloads/) หรือ
      ใช้ Jupiter Notebook ใน Anaconda navigator สามารถดาวโหลดจาก https://www.anaconda.com/download

2. **คลิก Repository**: คัดลอก repository นี้ไปยังเครื่องของคุณโดยการ clone
   https://github.com/KnotDevelop/Python-FinalProject.git

วิธีใช้แอพพลิเคชัน
---- All Commands ----
1. **command** - แสดงรายการคำสั่งทั้งหมดที่มีในโปรแกรม  
2. **add** - เพิ่มงานใหม่โดยระบุชื่อ, คำอธิบาย, และวันที่ครบกำหนด  
3. **view** - แสดงรายการงานทั้งหมดที่มีในระบบ พร้อมสถานะ (รอดำเนินการหรือเสร็จสิ้น)  
4. **done** - เปลี่ยนสถานะของงานเป็น "เสร็จสิ้น" โดยระบุ ID ของงาน  
5. **delete** - ลบงานออกจากระบบโดยใช้ ID ของงาน  
6. **search** - ค้นหาและแสดงรายการงานที่ต้องทำให้เสร็จก่อนถึงกำหนดเวลา  
7. **exit** - ออกจากโปรแกรม
